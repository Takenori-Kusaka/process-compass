このプロジェクトは、多くの人からの意見・アイデア・フィードバックをもらいながら育てていくことを前提にしています。

## フィードバックの方法

- **[GitHub Issues](https://github.com/Takenori-Kusaka/process-compass/issues)** — 意見、質問、内容の提案など何でも
- 各ページ右上の「ページを編集」リンクから、修正の Pull Request を直接提案可能
