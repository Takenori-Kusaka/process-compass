# research/ — 調査メモ置き場

このディレクトリは、サイトに公開する前の一次調査メモ・下書き・素材を置く作業領域です。
`src/content/docs/` 以下(公開ドキュメント)とは区別して管理します。

## 運用ルール

- ディープリサーチの結果、参考文献リスト、インタビューメモなどはまずここに置く
- 内容が体系化できた段階で `src/content/docs/` の該当フェーズ配下に清書して公開する
- ファイル名は `YYYYMMDD-topic.md` 形式を推奨

## 構成

```
research/
├── phase1/   # 現状調査の一次メモ
├── phase2/   # AIDLC・理想形調査の一次メモ
└── assets/   # 図の元データ(draw.io, PlantUML ソース等)
```
